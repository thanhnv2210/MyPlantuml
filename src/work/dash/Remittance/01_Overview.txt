1) HUB_State_Machine(THUNES, NETSCLICK)
How to add a new status on HUB's state machine:
- Adapter update:
	+) Add new status to HUB_STATUS
	+) Add the new mapping for StatusMapper: internal(com.singtel.ml.remittance.model.TransactionStatus) <-> external
	+) Add the new mapping for STATE_MACHINE for HUB


2) ML_State_Machine: com.singtel.ml.remittance.model.TransactionStatus
How to add a new status on ML state machine:
- Remit Service update:
	+) Step 1: get the existing status.
	+) Step 2: Event
		o Expected event: for example callback, or partner output response
			- check partner STATE_MACHINE for eligible status and update.
		o Unexpected event: Connection Issue, Response issue
			- Handle case by case on each step on errorHandler.
